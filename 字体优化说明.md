# 多语言字体优化方案

## 新字体方案

### 🎨 字体栈（优先级从左到右）

```css
font-family: 'Inter',           /* 1. 主字体 - 现代、专业 */
             'Noto Sans SC',    /* 2. 中文优化 */
             -apple-system,     /* 3. macOS系统字体 */
             BlinkMacSystemFont, /* 4. macOS渲染优化 */
             "Segoe UI",        /* 5. Windows 10+系统字体 */
             "PingFang SC",     /* 6. macOS中文 */
             "Microsoft YaHei", /* 7. Windows中文 */
             "Helvetica Neue",  /* 8. macOS备用 */
             Arial,             /* 9. 通用备用 */
             sans-serif;        /* 10. 系统默认无衬线 */
```

### 📖 字体详解

#### 1. Inter ⭐ (主字体)
**来源**: Google Fonts  
**特点**:
- ✅ 专为屏幕显示优化
- ✅ 优秀的数字和符号设计
- ✅ 完整的西里尔字母（俄语）支持
- ✅ 多种字重（400-800）
- ✅ 极佳的可读性
- ✅ 现代、专业的外观

**适用场景**: 英语、俄语、数字、符号

#### 2. Noto Sans SC ⭐ (中文字体)
**来源**: Google Fonts  
**特点**:
- ✅ Google和Adobe联合开发
- ✅ 完整的简体中文覆盖
- ✅ 与Inter风格一致
- ✅ 清晰易读
- ✅ 开源免费

**适用场景**: 简体中文

#### 3. 系统字体后备
为了性能和兼容性，保留系统字体作为后备：
- macOS: SF Pro（-apple-system）、PingFang SC
- Windows: Segoe UI、Microsoft YaHei
- 通用: Arial

### 🔧 字体渲染优化

已添加的CSS优化：

```css
-webkit-font-smoothing: antialiased;      /* macOS/iOS平滑渲染 */
-moz-osx-font-smoothing: grayscale;       /* Firefox macOS优化 */
text-rendering: optimizeLegibility;       /* 改善文本渲染质量 */
```

**效果**:
- ✅ 文字边缘更平滑
- ✅ 小字号更清晰
- ✅ 字母间距更协调
- ✅ 整体更专业

### 📦 字体加载策略

#### 优化加载性能

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
```

**优势**:
- ⚡ 预连接加速字体下载
- 📥 使用CDN（全球加速）
- 💾 浏览器自动缓存
- 🔄 自动更新

#### 字体显示策略

Google Fonts使用 `display=swap`：
- 先显示系统字体（避免闪烁）
- 字体加载后平滑切换
- 不影响首次加载速度

### 🔄 替代方案（如果需要）

如果Google Fonts访问受限或需要本地字体：

#### 方案A：只使用系统字体（无需CDN）
```css
font-family: -apple-system, BlinkMacSystemFont,
             "Segoe UI", "PingFang SC", "Microsoft YaHei",
             "Helvetica Neue", Arial, sans-serif;
```

#### 方案B：使用其他CDN字体
- **Nunito Sans** - 温暖、友好
- **Open Sans** - 经典、通用
- **Lato** - 优雅、现代

#### 方案C：本地托管字体
将字体文件下载到 `static/fonts/` 并使用 `@font-face`

### 📊 性能影响

#### 字体文件大小（估算）
- Inter (wght 400-800): ~150KB（压缩后）
- Noto Sans SC (wght 400-700): ~2.5MB（仅中文字符）

**总计**: ~2.65MB（首次加载，之后缓存）

#### 加载时间
- 快速网络（50Mbps）: <1秒
- 中速网络（10Mbps）: 2-3秒
- 慢速网络（2Mbps）: 10-15秒

**优化**: 
- ✅ 使用CDN（全球节点）
- ✅ 浏览器缓存（1年）
- ✅ 使用 `display=swap`（不阻塞渲染）
- ✅ 只加载需要的字重

### 💡 进一步微调（可选）

如果觉得某些地方字体还可以优化：

#### 标题使用特殊字体
```css
h1, h2, h3, h4, h5, h6 {
    font-family: 'Inter', sans-serif;
    font-weight: 700;
    letter-spacing: -0.02em; /* 稍微紧凑 */
}
```

#### 数字使用等宽字体
```css
.table td:has(> strong),
.stat-card h3 {
    font-variant-numeric: tabular-nums; /* 等宽数字 */
}
```

#### 代码和技术文本
```css
code, pre {
    font-family: 'JetBrains Mono', 'Consolas', monospace;
}
```
