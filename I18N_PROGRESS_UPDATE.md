## 📱 响应式优化

### Bootstrap 断点使用
```css
d-none d-xl-inline  /* 超大屏（≥1200px）显示 */
d-xl-none           /* 小于1200px 显示 */
d-none d-lg-inline  /* 大屏（≥992px）显示 */
```

### 导航栏显示逻辑
```
屏幕宽度 ≥ 1200px:  [图标] 完整文本
屏幕宽度 < 1200px:  [图标] 缩写文本
屏幕宽度 < 992px:   [图标] （悬停提示）
```
