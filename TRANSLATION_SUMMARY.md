## 🚀 使用方法

### 编译翻译
```bash
pybabel compile -d translations
```

### 重启应用
```bash
python app.py
```

### 切换语言
1. 点击导航栏右上角的语言切换器
2. 选择目标语言
3. 页面立即切换为对应语言

## 📂 关键文件

- `babel.cfg` - Babel配置
- `app.py` - Flask-Babel集成
- `translations/en_US/LC_MESSAGES/messages.po` - 英语翻译
- `translations/ru_RU/LC_MESSAGES/messages.po` - 俄语翻译
- `UI_CONSISTENCY_GUIDE.md` - UI一致性指南
- `common_translations.txt` - 常用术语速查表

## ⚡ 下次新增功能时

1. 在模板中使用 `{{ _('文本') }}`
2. 在Python中使用 `_('文本')`
3. 运行 `update_i18n.bat` 更新翻译
4. 编辑 `.po` 文件添加翻译
5. 运行 `compile_i18n.bat` 编译
6. 重启应用

## 🎯 建议

- **保持简洁**：UI文本尽量简短
- **使用图标**：减少文字依赖
- **测试三语**：每次修改都测试三种语言
- **优先俄语**：俄语文本最长，以其为基准
